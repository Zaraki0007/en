Real-Life RCE Example (OWASP Top 10)

Vulnerability: Remote Code Execution (RCE) via unsanitized input

Target: A web application that allows users to ping an IP address.

Vulnerable Code:

<?php $ip = $_GET['ip']; echo shell_exec("ping -c 4 " . $ip); ?>

Exploitation:

Since the input is directly concatenated into a system command, an attacker can inject additional commands.

Payload:

127.0.0.1; cat /etc/passwd

Impact:

By injecting `; cat /etc/passwd`, an attacker can read system files, execute arbitrary commands, and take control of the server.

Fix:

Use escapeshellarg() to sanitize input:

<?php $ip = escapeshellarg($_GET['ip']); echo shell_exec("ping -c 4 " . $ip); ?>

Conclusion

RCE is one of the most dangerous vulnerabilities in the OWASP Top 10. Always sanitize user input and avoid direct execution of user-controlled data.
